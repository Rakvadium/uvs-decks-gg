# Archive

Place **superseded** specifications, old API drafts, or historical phase documents here so the top-level [docs/README.md](../README.md) stays focused on current goals.

Completed backlog rows: [backlog-completed.md](./backlog-completed.md) (active list: [../BACKLOG.md](../BACKLOG.md)).

When you move a file into `archive/`, add a one-line note at the top of the archived file:

- **Status:** Archived as of YYYY-MM-DD — replaced by `../path/to/current-doc.md` (if applicable).

The **`_reference/docs/`** tree at the repository root remains a **separate example** documentation set from another product; it is not tcg-decks archive storage unless you explicitly decide to treat it that way.
